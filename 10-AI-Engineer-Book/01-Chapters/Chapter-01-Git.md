# Git

---

# مقدمه

قبل از اینکه برنامه‌نویسی را آغاز کنیم، باید با یکی از مهم‌ترین ابزارهای دنیای توسعه نرم‌افزار آشنا شویم.

این ابزار **Git** نام دارد.

تقریباً تمام شرکت‌های نرم‌افزاری دنیا، از شرکت‌های کوچک گرفته تا شرکت‌هایی مانند Google، Microsoft و OpenAI، از Git برای مدیریت پروژه‌های خود استفاده می‌کنند.

---

# Git چیست؟

**Git** یک سیستم مدیریت نسخه است.

نام انگلیسی:

**Version Control System (VCS)**

Git ابزاری است که تاریخچه تغییرات فایل‌های یک پروژه را ثبت، نگهداری و مدیریت می‌کند.

به کمک Git می‌توان:

- تغییرات پروژه را ثبت کرد.
- به نسخه‌های قبلی بازگشت.
- تغییرات را با یکدیگر مقایسه کرد.
- چند برنامه‌نویس را روی یک پروژه هماهنگ کرد.

---

# تشبیه

فرض کنید در حال نوشتن یک کتاب هستید.

هر بار که تغییر مهمی در کتاب ایجاد می‌کنید، از کل کتاب یک نسخه نگه می‌دارید.

اگر بعداً اشتباهی رخ دهد، می‌توانید به نسخه قبلی برگردید.

Git دقیقاً همین کار را برای پروژه‌های نرم‌افزاری انجام می‌دهد.

به همین دلیل می‌توان Git را «ماشین زمان پروژه» نامید.

---

# نکته مهندسی

Git فایل‌ها را فقط ذخیره نمی‌کند.

بلکه تاریخچه تغییرات فایل‌ها را مدیریت می‌کند.

این تفاوت، مهم‌ترین ویژگی Git است.

---

# واژه‌های جدید

| واژه | معنی |
|------|------|
| Git | سیستم مدیریت نسخه |
| Version | نسخه |
| Control | مدیریت |
| System | سامانه / سیستم |
| Repository | مخزن پروژه |
